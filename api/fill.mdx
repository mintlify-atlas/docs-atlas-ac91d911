title: "Fill"
description: "Linear and radial gradient fills for shapes and text."
---

`tvg::Fill` is the abstract base for `LinearGradient` and `RadialGradient`. Use gradients as fill or stroke for `Shape` and `Text` objects.

## ColorStop

```cpp
struct Fill::ColorStop {
    float   offset;  // position in gradient (0.0 = start, 1.0 = end)
    uint8_t r;       // red   (0–255)
    uint8_t g;       // green (0–255)
    uint8_t b;       // blue  (0–255)
    uint8_t a;       // alpha (0–255)
};
```

## Common Fill methods

```cpp
Result colorStops(const ColorStop* colorStops, uint32_t cnt) noexcept;
uint32_t colorStops(const ColorStop** colorStops) const noexcept;

Result spread(FillSpread s) noexcept;
FillSpread spread() const noexcept;

Result transform(const Matrix& m) noexcept;
Matrix& transform() const noexcept;

Fill* duplicate() const noexcept;
Type type() const noexcept;
```

**FillSpread values:**

| Value | Behavior |
|-------|---------|
| `Pad` | Extends the closest stop color beyond the bounds (default) |
| `Reflect` | Mirrors the gradient pattern |
| `Repeat` | Repeats the gradient pattern |

## LinearGradient

```cpp
static LinearGradient* gen() noexcept;

// Set gradient line from (x1,y1) to (x2,y2)
Result linear(float x1, float y1, float x2, float y2) noexcept;
Result linear(float* x1, float* y1, float* x2, float* y2) const noexcept;  // getter
```

**Example:**

```cpp
auto fill = tvg::LinearGradient::gen();
fill->linear(0, 0, 400, 0);  // horizontal gradient

tvg::Fill::ColorStop stops[3] = {
    {0.0f, 255,   0,   0, 255},  // red at start
    {0.5f, 255, 255,   0, 255},  // yellow at middle
    {1.0f,   0,   0, 255, 255},  // blue at end
};
fill->colorStops(stops, 3);
fill->spread(tvg::FillSpread::Pad);

shape->fill(fill);
```

## RadialGradient

```cpp
static RadialGradient* gen() noexcept;

// cx,cy = end circle center; r = end circle radius
// fx,fy = focal point (start circle center); fr = focal radius
Result radial(float cx, float cy, float r, float fx, float fy, float fr) noexcept;
Result radial(float* cx, float* cy, float* r,
              float* fx = nullptr, float* fy = nullptr, float* fr = nullptr) const noexcept;
```

For a simple centered radial gradient, align the focal point with the center and set `fr = 0`:

**Example:**

```cpp
auto fill = tvg::RadialGradient::gen();
fill->radial(400, 400, 150, 400, 400, 0);  // centered, no focal offset

tvg::Fill::ColorStop stops[2] = {
    {0.0f, 255, 255, 255, 255},  // white at center
    {1.0f,   0,   0,   0, 255},  // black at edge
};
fill->colorStops(stops, 2);

auto circle = tvg::Shape::gen();
circle->appendCircle(400, 400, 150, 150);
circle->fill(fill);
canvas->add(circle);
```

### Off-center focal point

Move the focal point to create directional lighting effects:

```cpp
// Focal point offset toward top-right
fill->radial(200, 200, 150, 230, 170, 10);
```

## Applying to strokes

```cpp
auto strokeGrad = tvg::LinearGradient::gen();
strokeGrad->linear(0, 0, 200, 0);
tvg::Fill::ColorStop stops[2] = {{0.0f, 255,0,0,255}, {1.0f, 0,0,255,255}};
strokeGrad->colorStops(stops, 2);

shape->strokeWidth(4.0f);
shape->strokeFill(strokeGrad);
```

## Gradient transform

Apply a `Matrix` to rotate or skew the gradient independently of the shape:

```cpp
tvg::Matrix m = {
    cosf(angle), -sinf(angle), 0,
    sinf(angle),  cosf(angle), 0,
    0,            0,           1
};
fill->transform(m);
```
