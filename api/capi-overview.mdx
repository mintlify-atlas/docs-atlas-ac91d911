ThorVG provides a C binding layer for use in C projects or when integrating with languages that have C FFI support (Python, Rust, Swift, etc.).

## Enabling the C API

Build ThorVG with the `capi` binding enabled:

```bash
meson setup builddir -Dbindings="capi"
ninja -C builddir install
```

Then include the C header:

```c
#include <thorvg_capi.h>
```

## Quick start in C

```c
#include <thorvg_capi.h>
#include <stdlib.h>

#define WIDTH  800
#define HEIGHT 600

int main() {
    // Initialize engine
    tvg_engine_init(4);

    // Create canvas
    uint32_t* buffer = malloc(WIDTH * HEIGHT * sizeof(uint32_t));
    Tvg_Canvas canvas = tvg_swcanvas_create();
    tvg_swcanvas_set_target(canvas, buffer, WIDTH, WIDTH, HEIGHT,
                            TVG_COLORSPACE_ARGB8888);

    // Draw a rectangle
    Tvg_Paint rect = tvg_shape_new();
    tvg_shape_append_rect(rect, 50, 50, 200, 150, 0, 0);
    tvg_shape_set_fill_color(rect, 100, 149, 237, 255);
    tvg_canvas_push(canvas, rect);

    // Render
    tvg_canvas_draw(canvas);
    tvg_canvas_sync(canvas);

    // Cleanup
    tvg_canvas_destroy(canvas);
    tvg_engine_term();
    free(buffer);
    return 0;
}
```

## Full C API reference

The complete C API is documented in [`src/bindings/capi/thorvg_capi.h`](https://github.com/thorvg/thorvg/blob/main/src/bindings/capi/thorvg_capi.h) in the repository.

<Tip>
  The C API mirrors the C++ API 1:1. If you understand the C++ API, the C functions follow the same pattern with `tvg_` prefix and snake_case naming.
</Tip>
