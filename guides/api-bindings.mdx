ThorVG's primary API is C++. A C binding layer is included in the repository, and community-maintained bindings exist for several other languages.

## C API

The C binding is built into ThorVG and enabled via a build flag:

```bash
meson setup builddir -Dbindings="capi"
ninja -C builddir install
```

Include `thorvg_capi.h` in your C project:

```c
#include <thorvg_capi.h>
```

See the [C API Reference](/api/capi-overview) for full documentation.

## Web / npm

ThorVG is available as an npm package for web applications:

```bash
npm install @thorvg/lottie-player
```

The WebAssembly build supports both SVG and Lottie rendering in the browser using the WebGL or WebGPU backend.

## Python example

```python
import thorvg as tvg

tvg.init(0)

canvas = tvg.SwCanvas()
buf = bytearray(800 * 600 * 4)
canvas.target(buf, 800, 800, 600, tvg.ColorSpace.ARGB8888)

shape = tvg.Shape()
shape.append_rect(50, 50, 200, 150, 10, 10)
shape.fill(100, 149, 237, 255)
canvas.push(shape)

canvas.draw()
canvas.sync()

tvg.term()
```
