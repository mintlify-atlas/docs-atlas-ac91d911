ThorVG is designed for portability across all major platforms and has been adopted in production by many projects.

## Supported platforms

| Platform | Notes |
|----------|-------|
| Linux | Full support; primary development platform |
| Windows | Full support; Visual Studio and MSYS2 builds |
| macOS | Full support; Homebrew available |
| iOS | ARM builds supported |
| Android | Kotlin bindings available |
| Web (WASM) | WebGL and WebGPU backends; npm packages available |
| Tizen | Integrated as the official vector graphics engine |
| Embedded / RTOS | Runs on ESP32, Tizen RT, and other MCUs |

<Note>
  The core library (~150KB) runs on microcontrollers such as the ESP32 with limited RAM and storage. For MCU targets, disable Lottie expressions and unused loaders to minimize binary size.
</Note>

## Cross-compilation

The `cross/` directory in the repository contains sample cross-compilation files for various targets. Use `--cross-file` with Meson:

```bash
meson setup builddir --cross-file cross/your-target.txt
```
